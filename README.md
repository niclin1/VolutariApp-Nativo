# VolutariApp-Nativo

A React Native app built with [Expo](https://expo.dev).

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- [Expo Go](https://expo.dev/go) app on your iOS or Android device, or an emulator/simulator

### Install dependencies

```bash
npm install
```

### Run the app

```bash
# Start the Expo development server
npm start

# Run on Android
npm run android

# Run on iOS (macOS only)
npm run ios

# Run on web
npm run web
```

## Project Structure

```
├── App.js          # Root component
├── index.js        # Entry point (registers root component with Expo)
├── app.json        # Expo configuration
├── assets/         # Static assets (icons, splash screen, etc.)
└── package.json    # Project dependencies and scripts
```
